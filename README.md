## Описание

GitHub User Finder - это десктопное приложение с графическим интерфейсом для поиска пользователей на платформе GitHub. Приложение использует официальное GitHub API для получения информации о пользователях и позволяет сохранять понравившихся пользователей в избранное.

### Основные возможности

- 🔍 Поиск пользователей GitHub по имени пользователя
- 📋 Отображение подробной информации о пользователе
- ⭐ Добавление пользователей в избранное
- 💾 Сохранение избранных пользователей в JSON файл
- 📜 История поиска
- 🖥️ Интуитивно понятный графический интерфейс
